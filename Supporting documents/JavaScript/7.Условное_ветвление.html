Когда нужно выполнить действия в зависимости от условий мы используем инструкцию if и условный оператор ? (или же вопросительный знак) 1) Инструкция if if (...) вычисляет условие в скобках и, если результат true - выполняет блок кода. let year = prompt('В каком году была опубликована спецификация ECMAScript-2015?', ''); if (year == 2015) alert( 'Вы правы!' ); Если мы хотим выполнить более одной инструкции, то нужно заключить блок кода в фигурные скобки (лучше в принципе после if всегда ставить их для удобочитаемости): if (year == 2015) { alert( 'Правильно!' ); alert( 'Вы такой умный!' ); } Преобразование к логическому типу: Инструкция if (...) вычисляет выражение в скобках и преобразует результат к логическому типу. Как мы помним: Число 0, пустая строка "", null, undefined, NaN - становятся false. (поэтому их называют ложными значениями) Остальные становятся true - правдивыми То есть, код при таком условии никогда не выполнится: if (0) { ... } //0 if falsy А при таком выполнится: if (1) { ... } //1 is truthy Также можно присвоить переменной само условие if: let condition = (year == 2015); if (condition) { ... } _____________________________________________________ Блок else if может содержать необязательный блок else Он выполняется, когда условие ложное let year = prompt('В каком году была опубликована спецификация ECMAScript-2015?', ''); if (year == 2015) { alert( 'Да вы знаток' ); } else { alert( 'А вот и не правильно!' ); //любое значение, кроме 2015 } НЕСКОЛЬКО УСЛОВИЙ else if let year = prompt('В каком году была опубликована спецификация ECMAScript-2015?', ''); if (year < 2015) { alert( 'Это слишком рано...' ); } else if (year > 2015) { alert( 'Это поздновато') } else if (year == 2015) { alert( 'Правильно!' ); } _________________________________________________________________________ Условный оператор "?" Иногда нужно определить переменную в зависимости от условия let accesAllowed; let age = prompt('Сколько вам лет?', ''); if (age > 18) { accessAllowed = true; } else { accessAllowed = false; } alert(accessAllowed); ОДНАКО! Оператор "Вопросительный знак" позволяет сделать код более коротким. Оператор ? = "тернарный", так как этот оператор, единственный оператор, который имеет три аргумента. Синтаксис: let result = условие ? значение1 : значение2; СНАЧАЛА вычисляется условие: если оно истино, тогда возвращается значение1, иначе - значение 2. Например: let accesAllowed = (age > 18) ? true : false; Оператор ? имеет низкий приоритет, поэтому будет выполняться после >, технически скобки можно опустить В примере выше вы можете избежать использования оператора вопросительного знака ?, т.к. сравнение само по себе уже возвращает true/false: // то же самое let accessAllowed = age > 18; Несколько операторов „?“ Последовательность операторов вопросительного знака ? позволяет вернуть значение, которое зависит от более чем одного условия. Например: let age = prompt('Сколько вам лет?', ''); let message = (age < 18) ? 'Привет, малой' : (age < 100) ? 'Добрый день' : (age > 100) ? 'Дарова плесень'; alert( message ); ИЛИ МОЖНО ТАК let company = prompt('Какая компания создала JavaScript?', ''); (company == 'Netscape') ? alert('Верно!') : alert('Неправильно.'); Не рекомендуется использовать оператор вопросительного знака таким образом.
