Однострочные комментарии: Начинаются с: // Многострочные: начинаются с: /* заканчиваются: */ ТО ЕСТЬ как в CSS ///////////////////////////////////////////////////// Для активации современного режима: Прописываем в начале скрипта: "use strict" - отменить нельзя + пишем на первой строчке. __________________________________________________ Переменные - можно перезаписать (let) Используются для хранения информации. Это именнованное хранилище для данных. (например, для хранения товаров, посетителей и других данных) let message; - объявляем переменную message message = 'Hello'; - сохраняем строку 'Hello' в переменной с именем message В итоге: let sms; sms = 'Привет!'; alert(sms); - показывает содержимое переменной Но лучше сразу объявить и присвоить: let sms = 'Привет!'; alert(sms); А также мы можем объявить сразу несколько переменных в одной строке (хотя лучше каждую переменную с новой строки) let user = 'John', age = 25, message = 'hello'; ============================================= Чтобы использовать константу (её нельзя перезаписать) - лучше использовать const вместо let: const myBirthday = '14.03.1982'; const COLOR_RED = "#F00"; const COLOR_GREEN = "#0F0"; const COLOR_BLUE = "#00F"; const COLOR_ORANGE = "#FF7F00"; // ...когда нам нужно выбрать цвет let color = COLOR_ORANGE; alert(color); // #FF7F00 ___________________________________________________
